Welcome to Roland Plotter Web UI

This app brings a modern interface to your Roland DXY plotter via the USB-to-serial CH340 adapter, where you can create and edit artwork, preview motion, and export or stream HPGL code ready for your next creative journey.

Connect to the plotter over USB serial and send manual commands.
Draw, import, preview, and simulate plot files before running them.
Generate patterns, handwriting, and image-to-vector artwork inside the interface.
